Vibration Profiling of ball-screw

Components: Accerlometer(ADXL), Encoder, microSD card shield, Arduino.

DataAquisation folder contain code for reading vibration data and distance(position) with the help of ADXL, Encoder
DataAquisationAndSaving folder contain code for reading as well as saving the above data into SD card.
SDcardTest contains code for testing SDCard.
